# Phishing Email Detection using Machine Learning

## Project Description

This project detects whether an email is **Safe** or **Phishing** using Machine Learning.

## Technologies Used

- Python
- Scikit-learn
- Pandas
- NumPy
- Flask
- HTML
- CSS
- Joblib

## Features

- Email Classification
- TF-IDF Feature Extraction
- Logistic Regression Model
- URL Detection
- Confidence Score
- Flask Web Interface
- Confusion Matrix

## Folder Structure

```
dataset/
model/
results/
static/
templates/
```

## How to Run

```bash
pip install -r requirements.txt
python app.py
```

Open:

```
http://127.0.0.1:5000
```

## Output

- Safe Email
- Phishing Email
- Confidence Score
- URL Count
